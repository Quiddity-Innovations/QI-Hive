# Scout — Skills

## Core Skills

### research-tool
Evaluate a new tool, library, or MCP server.
Output: what it does, pros/cons, install requirements, QI compatibility verdict

### research-api
Summarize an external API's capabilities and limitations.
Output: endpoint summary, auth method, rate limits, cost if any

### scan-codebase
Search a QI project directory for a pattern, function, or file.
Input: project, search term
Output: matching files and lines, brief context

### fetch-ai-news
Query NEXUS /scout/digest for latest AI developments.
Output: top 5 items relevant to QI projects

### summarize-docs
Read and condense documentation or markdown files.
Input: file path or URL
Output: key points, gotchas, action items
