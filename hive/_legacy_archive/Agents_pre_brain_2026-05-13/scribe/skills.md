# Scribe — Skills

## Core Skills

### session-summary
Create a Word document summarising a session.
Output: .docx file at C:\{project}\DOCUMENTATION\Session Summaries\{Project}_Summary_{YYYY-MM-DD_HHMM}.docx

### update-implementation-log
Append a dated entry to the project's Implementation Log.
Input: what was built/changed this session

### update-meeting-minutes
Append a dated entry to Meeting Minutes.
Input: decisions made, people involved, next steps

### update-version-history
Append an entry to Version History when code changes.
Input: version bump (patch/minor/major), change summary

### update-latest-md
Rewrite LATEST.md with the current session handoff.
Output: C:\Claude\Session Summaries\LATEST.md

### write-docstring
Add docstrings to Python functions/classes.
Input: file path, function name
Output: docstring written in Google or NumPy format

## Trigger Phrases
- "document ..."
- "write up ..."
- "save the session"
- "update the logs"
- "write the summary"
