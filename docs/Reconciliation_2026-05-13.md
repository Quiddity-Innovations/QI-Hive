## What shipped (evidence-only)

### 1. Hive Inspector — Standards Enforcer agent (2026-05-09)
Built to end the silent-drift problem (projects accumulating uncommitted work,
stale docs, broken services without anyone noticing). Lives at
`engine/hive/inspector/inspector.py`.

- **Two modes:** `fast` (cheap file/JSON/hook checks, used every 4h and by the
  real-time poller) and `deep` (fast + HTTP probes + git log scan + NSSM
  probes, used nightly).
- **Three actions per finding:** auto-fix (deterministic), dispatch (ambiguous
  — files a dispatch for human or agent), or log-only.
- **Database:** every check writes to `brain.compliance_log` with severity,
  action_taken, fix_action, and dispatch_id.
- **Brain API surface** (`engine/brain/api.py` +108 lines):
  - `POST /api/compliance/scan` — run a scan
  - `GET /api/compliance/recent` — last N log entries
  - `GET /api/compliance/status` — current red/yellow/green per project
- **Dashboard UI:** `/compliance` page renders the Inspector results.
